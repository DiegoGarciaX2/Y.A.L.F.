# Y.A.L.F.
Repositorio para sigue lineas
## La disposicion de los pines de Y.A.L.F. es:
- SLC->A5
- SDA->A4
- Motor A(derecha):
  - C1->7
  - C2->8
  - int1->6
  - int2->5
- Motor B(izquierda):
  - C1->2
  - C2->3
  - int3->11
  - int4->10
  
#### El encoder de los motores es de siete pulsos
